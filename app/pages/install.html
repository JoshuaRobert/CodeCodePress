Installation

  1. Go to where you want your project to be in terminal, and type

    git clone https://github.com/nonlinear/codecode

  2. Rename

    codecode

    folder to your project name.
  3. Go to your project folder on terminal and type

    npm install

    It will take a while.

  4. Create a new repository on github, copy the HTTPS: url, then type:

    git remote rename origin upstream

    git remote add origin (paste HTTPS here)

    git push origin master

  5. Clean up code with

    gulp start

  6. Move

    macros

    folder to

    Library/Application Support/Sublime Text 2 (or 3)/Packages

    .

Usage

  1. On terminal, go to project and type

    gulp

    A browser window will open, and refresh anytime your files are edited.

  2. Files are on

    app/pages

  3. Templates are on

    app/templates

  4. Personalize base variables at

    app/sass/style.sass

  5. Edit project CSS rules at

    app/sass/project/layout.sass

  6. Push your changes at will, and when you're ready to publish it, type

    gulp publish

    unCSS will remove unused rules, and gh-pages will publish it at www.your-github-username.github.io/project-name
